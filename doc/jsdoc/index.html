Installation

You need to clone the git repository and copy directory to Canopsis path

$ git clone https://git.canopsis.net/canopsis-ui-bricks/brick-rrule-editor.git
$ cp -r brick-rrule-editor $CANOPSIS_PATH/var/www/canopsis

Then, you need to import specific schemas

$ su - canopsis
$ cp $CANOPSIS_PATH/var/www/canopsis/brick-rrule-editor/schemas/* $CANOPSIS_PATH/etc/schema.d
$ schema2db

Then, you need to enable the brick

$ su - canopsis
$ webmodulemanager enable brick-rrule-editor

You can see enabled bricks

$ su - canopsis
$ webmodulemanager list
[u'core', u'uibase', u'monitoring', ..., **u'brick-rrule-editor'**]
